# Express, TypeORM and Typescript Example project

This project is a basic starter project on how to use Express and TypeORM together using Typescript.

The project is using right now SQLite as its DB engine.

## Install

```cmd
    $ npm install
```

## Run Project

```cmd
    $ npm run watch
```
